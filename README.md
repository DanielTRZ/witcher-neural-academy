# witcher-neural-academy

Wiedźmińska Akademia – Jak działają sieci neuronowe?

Opis

„Wiedźmińska Akademia” to interaktywna, edukacyjna aplikacja w HTML/CSS/JS, która w klimacie Wiedźmina pokazuje działanie sieci neuronowych.

Animowane neurony z pulsującymi sygnałami

Bestiariusz sztucznej inteligencji (Dropout, Overfitting, Backpropagation)

Mini tryb treningu sieci z efektownymi opisami kroków

Wybór funkcji aktywacji (ReLU, Sigmoid, Tanh, Linear)
